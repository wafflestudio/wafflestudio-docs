# wafflestudio-docs

readthedocs로 만들어진 와플스튜디오 세미나 자료입니다.

https://wafflestudio.readthedocs.io

## 개발 환경 설정

`pip install -r requirments.txt`

## 로컬 환경에서 빌드하는 법

docs 경로로 들어가서 다음과 같은 명령어를 입력한다.
`sphinx-build -b html . ./_build`
